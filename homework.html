Homework policy

Timeframe

You are given one week to do each homework assignment. Homework is due before class starts one week from when it is assigned. For example, if an assigment is given on a Thursday, it's due at 6:30pm the following Thursday.

Submitting Homework

Homework is "turned in" by putting it into GitHub. You'll probably want to create a folder called "homework" to put every assignment in as there will be many!

  1. Create a folder in your GitHub repo for each new assignment.
  2. Put all files related to that homework assigment in the new folder (html/css/js/images).
  3. Add a link to your homework on your main GitHub home page.
  4. Commit and sync your changes to GitHub.

Grading

Each homework assignment is worth a total of 6 points.

You get up to 3 points if your work functions correctly and looks correct. The other 3 points are for the quality. Style is important as a developer and this includes using proper formatting, indenting, and syntax.
