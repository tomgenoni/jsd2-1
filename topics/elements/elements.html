JSD2: Topics: Elements

Elements

Objectives

You should be comfortable with:

  1. Getting an element out of the DOM and saving it into a variable, with querySelector()
  2. Creating a new element, with createElement()
  3. Inserting the new element into the DOM tree, with appendChild()
