Switch Statements

Notes

  • Used for branching code
  • A if/else is used for more advanced conditional logic checks
  • A switch is used for checking the value of one variable against many possible values.
  • It's usually used with a string or number
  • Has an optional "default" case that is similar to the "else" of an if/else structure.

Syntax / Anatomy

switch(varname) {
	case "Admin":
		...
		break;
	case "Superuser":
		...
		break;
	default:
		...
}

Fall through

This syntax combines multiple cases into one code branch.

Fall-through example:

switch(varname) {
	case "Admin":
	case "Superuser":
		...
		break;
	case "User"
		...
		break;
	default:
		...
}
