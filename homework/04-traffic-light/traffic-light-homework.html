Traffic Light

Overview

  • Your assignement is to make a working traffic light. By using the control buttons at the top of the page, the light should be able to be changed from red to green to yellow.
  • Just as with the examples in class, you'll be adding event listeners to the buttons. There will be three corresponding event handler functions. What you have to figure out is how to "control the lights."
  • The new concept for this homework assignment is changing class names on an Element. This is done using the ClassList API. The ClassList API is a pretty simple object that has methods to add, remove, and toggle CSS classes on an Element.

Task

  1. Review the HTML that is given. Really get a feeling for the structure of the DOM. Play around with the Elements tab in the web inspector.
  2. Analyize the CSS that is given. Try to manually set the light color by adding a class to the traffic-light Element in the static html file and reloading the page. One by one, add the stop, slow, etc classes to the traffic-light in the HTML file, reload the page, and see what happens. Super important to do this before you write even one line of code!
  3. The JS file has already been linked for you, it's called trafficLight.js. In there, write the structure, events, and event handlers to make your traffic light come alive!
  4. The Event Handler functions will have to make heavy use of ClassList API to add and remove CSS classes from the traffic-light div.
  5. Once that is complete, try to enable a 'caution' mode where the yellow light flashes. Hint: use a Timer.

Given

  • The mockup image of a Traffic Light attached to this assignment.
  • All the HTML and CSS is done for you.

If you get stuck, remember the processes we've used in class:

  • Getting Elements with document.querySelector()
  • Saving Elements in variables with var using assignment (=)
  • Making an Element listen for a click Event with addEventListener()
  • Grouping together code with functions. Remember, to use an Event Handler function for each Event Listener.
  • New: Changing an Element CSS class using the ClassList API.

Don't stress

There are many solutions. Run with whatever makes the most sense to you. Don't worry about the "best" way. Every class I end up changing up what I think is the best way. I look at my code from last time, and think: that's kinda weird, I could do better.

Minimum Requirements

Clicking "Stop" should make it red, "Slow" should make it yellow, and "Go" should make it green.

Tiny Bonus

Add the ability to flash the yellow light like a caution light. Everyeone should strive for this bonus. It's not to hard and shows off your skillz with Timers.

Super Crazy Bonus

Add a "run" button that automatically cycles through the red/green/yellow lights. Design the timing for each light. A typical light cycle is 120 seconds. But you may want to shorten that to like 12 for demo purposes.
