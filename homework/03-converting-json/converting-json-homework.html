Overview

This exercise is to get you more comfortable converting JSON from strings and parsing data out of a block of JSON data. There are four challenges. Complete each of the challenges, typing the JS code to solve it directly into the file.

Given

  • A zip file with a project template folder structure (download above).
  • HTML/CSS: no html/css - this is a console only assignment
  • A JavaScript file, "convertingJson.js", that contains a string of JSON data.

Minimum Requirements

1. Convert the string `jsonString` to a Javascript Object and store it in a variable called `flicker`

2. Iterate (this means 'use a loop') and do a console.log of each of the item titles with the date the photo was taken. The first console.log should return the following:
"What a surprise in lyon. 2016-02-23T06:47:30-08:00"

3. Create a new array called 'links'. Iterate through the items array in flicker and store the the links to each item in the 'links' array.

4. Convert the 'links' array into a JSON string. Do a console.log of that JSON string, copy the results from the terminal and paste them into the input box at: http://jsonlint.com

Did it create a valid JSON string?
