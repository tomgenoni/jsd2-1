Assignment:

Your task is to write four functions that will make three calculators work.

Download the zip file with the starter code, you'll see a formulas.html file. Run the formulas.html file and determine what is working and what is not working.

You will be placing your code in the formulas.js file.

As always, link the final results (all three files) to your own Github page.

Here are the formulas:

  • Celcius to fahrenheit: celcius * (9/5) + 32
  • Fahrenheit to celcius: (fahrenheit - 32) * 5/9
  • Radius to circumference: 2 * Math.PI * radius
  • Pythagorean theorem: Math.sqrt((a * a) + (b * b))
